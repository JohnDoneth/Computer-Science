CHANGELOG

Changelog

v2.11.13 - 12/14/18

  • Fixes EMS starting on a "Default" IP address instead of localhost/127.0.0.1
  • Added icons in the match results screen

v2.4.0 - 8/12/18

  • Ability to delete all processes within EMS
  • Support for a sponsor screen in the audience display
  • On application boot, during the splash screen an "EEPERM" error might show, just close EMS and restart the program as administrator.

v2.2.1 - 8/10/18

  • Match review/score updating in the 'Match Review' tab
  • Field controlled in the event configuration will limit what matches show up in match review/match play.
  • Font size in Match Play screen for fgc_2018 was lowered.

v2.0.0 - 8/7/18

  • Initial release of EMS v2.0. Node.js v8.0 or later must be installed to run.
