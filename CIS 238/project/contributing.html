Contributing

The Event Management System is an open source project. If you wish to contribute to it or another project by The Orange Alliance, check out our GitHub repository.

The Event Management system was built upon the following technologies:

  • Node.js
    • Express.js
    • Electron.js
  • React.js
  • TypeScript
  • SQLite3

If you have any experience in these, please consider contributing!
