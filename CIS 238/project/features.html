Event Management System

FEATURES

Features

The Event Management System is highly versatile, and can suit the needs of your event. For a list of event types and games supported by EMS, visit the Event Type & Games page. The following are a list of its most impressive features.

Customized Displays

EMS supports customized displays. You can swap which sides the scores are on, whether to use real-time updating, or set a team's identifier.

Each display also features special, smooth animations for score results and during match play. Web technologies such as CSS3 and WebGL help fuel fast, clear displays.

Advanced Event Administration

At your very fingertips, you can manage users on your field network and create customized match schedules with any number of fields concurrently on one division.

Event and match databases can be encrypted with a specialized cipher, as well as a special password for certain actions within EMS.

Real-Time Scoring

As a part of the EMS suite, the referee scoring application can be used to provide real-time scoring to the audience during the match.

If this feature is not desired, it can be disabled in the settings of the EMS desktop application.

Public API

During an event, all of its data is passed around through an API that EMS uses itself. This API (if enabled) is public, and can be hacked on for custom applications.

Ask your event administrator for special access to this API. If granted, be sure to check out the documentation!
