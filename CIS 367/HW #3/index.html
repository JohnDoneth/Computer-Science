Super Smash Bros. Abstract

By Kyle Flynn (~4 hours)

Scene Background Information

This WebGL 2D render is of a few video game assets from the new Nintendo Switch game, Super Smash Bros. Ultimate. Unfortunately my WebGL skills are not yet up to par, but this is a nice framework for a scene that introduces the new character, the Piranha Plant from the Super Mario Bros. series. The red sphere is the logo for the game, and the other object is of the character, Piranha Plant.

Scene Marked Sketch

The marked sketch may seem a little hard to understand, but it's divided into 3 parts. Blue markings represent quad polygons using TRIANGLES_STRIP. Gold markings represent circles, ellpises, or cones (parts of circles) that can be drawn using TRIANGLES_FAN. And finally, Pink markings represent actual triangles that can be drawn using TRIANGLES.

NOTE: Not shown are two outlines. The first outline is all along the logo circle object on the right, this will be done using LINE_STRIP. Secondly, there will be an outline along the bottom of the character for clarification using LINES.

Scene WebGL Result

The most intriguing/challenging aspect of this rendering is the mouth opening/cone shape in the Piranha plant. Since circles are within the polar plane and we are in the cartesian coordinate system, it's a little hard to take a chunk of a circle. However, with a little experimentation and math it wasn't that difficult, but the started code provided lots of versatility. For example, the brown shadow at the bottom was created as an ellipse that takes a xMax and yMax parameter.

The depth effect was used a little bit differently in this scene. The depth effect was used to create layering, such as for the teeth to overlay on top of the mouth cone opening, which layers on top of the piranha plant head.
